Use xinit. To make startx run at startup after login automatically, put the following script in ~/.bash_profile:
if [ -z "${DISPLAY}" ] && [ "${XDG_VTNR}" -eq 1 ]; then
  exec startx
fi


Install google-chrome: yay -S google-chrome

Configure xinitrc:
cp /etc/X11/xinit/xinitrc ~/.xinitrc
nano ~/.xinitrc
Replace lines from the end where we spawn programs like xterm, xclock etc.. Basically after the end of last if... replace with the contents indicated here in the sample .xinitrc file
Disable a function in the compositor: sudo nano /etc/xdg/picom.conf and disable vsync. This is required for VM.. haven't tested it though
Pull up browser and set background with nitrogen
Remember to set the wallpaper otherwise nitrogen --restore doesn't work
